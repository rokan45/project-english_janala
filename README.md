# 📚 Vocabulary Learning App

A dynamic vocabulary learning web application that allows users to explore words by levels, view meanings, hear pronunciations, and save favorite words.


## 🚀 Live Demo

https://rokan45.github.io/project-english_janala/


## ✨ Features

* Levels & Lessons
* Vocabulary Cards
* Active Level Highlighting
* Search Functionality
* Vocabulary Details Modal
* Save Words Feature
* Voice Pronunciation
* Loading Spinner
* Error Handling


## 🛠️ Technologies Used

* **HTML5** – Structure
* **Tailwind CSS & DaisyUI** – UI design
* **JavaScript (Vanilla)** – Logic and interactivity
* **API Integration** – Dynamic data handling


## 🧠 Learning Objectives

This project helped in:

* Working with APIs (fetch, async/await)
* DOM manipulation and dynamic UI rendering
* State management (active buttons, saved items)
* Handling edge cases and user experience
* Implementing browser features (speech synthesis)
